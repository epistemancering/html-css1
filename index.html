HTML & CSS 1

Summary

In this lab exercise, I created this page using HTML & CSS. There was nothing to download as I started from scratch.

Instructions
  1. I decided what I wanted to make my page about - it could have been about a hobby, a book, a movie, an event, a place, or anything else I could think of
  2. I sketched out a basic layout (I could have done this anywhere, I won't be submitting it)
  3. I created a folder for my project, inside the folder, I created an index.html file and a styles.css file
  4. In the HTML file, I set up my basic structure - doctype, head, body
  5. I started coding! The LMS recommended breaking down my sketch into small parts. I did all of the HTML and then all of the CSS before moving on to the next part. I didn't worry if my code was messy or repetitive right then - I was learning!

Intermediate

Once I created my basic page, I did some HTML & CSS research and saw what I could add! We'll cover classes & ids, the position, display, & flexbox properties, as well as semantic HTML later in this unit; so I used my time in this lab to look at other topics. W3Schools and MDN are both great reference sites for HTML & CSS.
